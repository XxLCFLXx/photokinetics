# PINN 案例：非均匀消光系数 κ(z) 分布反演

**日期**：2026-07-16
**作者**：Cogito Lin
**目标**：用 PINN + photokinetics 可微物理引擎，从稀疏时空温升观测 T(z,t) 反演非均匀消光系数分布 κ(z)，证明可微物理在逆问题中的"不可替代性"。

---

## 1. 问题动机与不可替代性论证

### 1.1 物理场景

实际工程中，材料内部消光系数 κ 经常是非均匀的：
- 镀膜基底界面（折射率渐变层）
- 缺陷层 / 杂质聚集区
- 生物组织（皮肤分层、血管吸收差异）
- 复合材料界面

此时解析解 `calc_photothermal_timed` 完全失效——它假设 κ 是常数。

### 1.2 三类方法的对比

| 方法 | 能否处理非均匀 κ(z) | 能否做参数反演 | 备注 |
|------|:---:|:---:|------|
| 解析解 (Carslaw-Jaeger) | ❌ | ❌ | 假设 κ 常数，根本无法表示非均匀 |
| FDTD 数值模拟 + scipy 反演 | ✅ | ❌ | 每次反演迭代需重跑 FDTD（小时级），参数反演不可行 |
| **PINN + 可微物理** | ✅ | ✅ | 神经网络参数化 κ(z)，PDE 残差作为可微损失，单次优化即可反演 |

**核心论点**：photokinetics 的可微物理引擎让 PDE 残差 ∂T/∂t − D·∂²T/∂z² − q(z)/(ρCp) 端到端可微，这是 PINN 能同时反演 κ(z) 分布 + 重建 T(z,t) 场的物理基础。

### 1.3 与已有 3 个案例的关系

| 案例 | 反演目标 | 解析解可用性 | 可微物理作用 |
|------|---------|:---:|------|
| 1. 含时光热反演 | 标量 κ, D | ✅ 可用 | autograd 精确梯度打破 κ-D 简并 |
| 2. 黑体测温 | 标量 T | ✅ 可用 | 对数域 loss + autograd |
| 3. 光镊鲁棒设计 | 标量设计参数 | ✅ 可用 | 多目标可微优化 |
| **4. PINN 非均匀 κ(z)** | **函数 κ(z)** | **❌ 不可用** | **PDE 残差可微，参数化 κ(z) 场** |

PINN 案例是可微物理"不可替代性"的最强证据：前 3 个案例 autograd 只是"比 scipy 更好"，PINN 案例是"scipy 根本无法处理"。

## 2. 物理模型

### 2.1 控制方程

1D 含源热传导方程（半无限域 z ≥ 0）：

```
∂T/∂t = D · ∂²T/∂z² + q(z; κ) / (ρ Cp)
```

其中：
- D = k_th / (ρ Cp)：热扩散率（已知，材料手册查得）
- ρ, Cp：密度、比热（已知）
- q(z; κ)：光热源项，**依赖未知 κ(z)**

### 2.2 非均匀光热源

标准比尔-朗伯定律在非均匀介质中推广为：

```
dI/dz = -α(z) · I(z),    I(0) = I₀
→ I(z) = I₀ · exp(-∫₀ᶻ α(z') dz')

α(z) = 4π κ(z) / (n λ)
q(z) = α(z) · I(z)
```

**关键**：α(z) 由神经网络 κ_φ(z) 给出，I(z) 通过数值积分（cumtrapz）从 α(z) 计算，整个链条可微。

### 2.3 边界与初始条件

- IC: T(z, 0) = 0（初始零温升）
- BC1 (z=0): ∂T/∂z|_{z=0} = 0（绝热表面，无热流损失）
- BC2 (z→∞): T(z→∞, t) = 0（半无限域，远场零温升）

### 2.4 真值 κ(z) 设计

模拟"聚合物基底内 5μm 深处有一层强吸收缺陷层"：

```
κ_true(z) = κ₀ · (1 + A · exp(-(z - z₀)² / (2σ²)))

材料：聚合物（PMMA-like，含染料）@ 532nm
  n = 1.49         # 折射率
  κ₀ = 0.005       # 基底消光系数（较弱吸收）
  ρ = 1190 kg/m³   # 密度
  Cp = 1420 J/(kg·K)
  k_th = 0.2 W/(m·K)
  D = k_th/(ρCp) = 1.18e-7 m²/s   # 热扩散率（比硅小 ~1000 倍）

缺陷层参数：
  A  = 3.0         # 缺陷层峰值放大倍数（峰处 κ = 4κ₀）
  z₀ = 5e-6 m      # 缺陷层中心深度（5μm）
  σ  = 2e-6 m      # 缺陷层宽度（2μm）
```

**材料选择理由**：硅 D=9.08e-5 m²/s 太大，1D 显式 FD 受 CFL 限制 (D Δt/Δz² ≤ 0.5) 需要极小时间步。聚合物 D 小 1000 倍，配合隐式格式可以稳定求解。这是工程取舍，不影响 PINN 方法的普适性。

物理意义：在 5μm 深处有一个 κ 高 4 倍的薄层，对应材料内部约 6μm 厚度（FWHM ≈ 2.355σ ≈ 4.7μm）的强吸收区，模拟缺陷层 / 染料聚集 / 界面吸收增强等场景。

## 3. 数据生成（真值 T(z,t)）

### 3.1 有限差分求解器（隐式后向欧拉）

由于解析解无法处理非均匀 κ(z)，用 1D **隐式后向欧拉**有限差分生成"真值" T(z,t)：

```
离散化：
  (T^{n+1}_i - T^n_i) / Δt = D · (T^{n+1}_{i+1} - 2 T^{n+1}_i + T^{n+1}_{i-1}) / Δz² + q_i / (ρCp)

整理为线性系统：
  (I - D Δt / Δz² · A) · T^{n+1} = T^n + Δt · q / (ρCp)

其中 A 是三对角拉普拉斯算子：
  A = tridiag(1, -2, 1)
  配合 Neumann BC at z=0 (T_{-1} = T_1) 和 Dirichlet BC at z=L (T_L = 0)
```

**为什么用隐式而非显式**：
- 显式 Euler 受 CFL 限制 D Δt/Δz² ≤ 0.5
- 聚合物 D=1.18e-7 m²/s, Δz=0.1μm → Δt ≤ 0.5·(1e-7)²/1.18e-7 ≈ 4.2e-8 s
- 模拟 0.1s 物理时间需要 Nt ≈ 2.4e6 步，太慢
- 隐式后向欧拉无条件稳定，可以任意细化 Δz

用 `scipy.linalg.solve_banded` 高效求解三对角系统（O(Nz) 复杂度）。

**网格**：
```
z ∈ [0, 30μm],  Nz = 300 (Δz = 0.1μm)  # 能分辨 2μm 缺陷层（FWHM ≈ 4.7μm）
t ∈ [0, 0.1s],  Nt = 1000 (Δt = 1e-4s)
```

**验证**：均匀 κ 下，FD 解与 `calc_photothermal_timed` 解析解误差应 < 5%（测试用）。

### 3.2 光强 I(z) 计算（非均匀）

```
dI/dz = -α(z) · I(z),  I(0) = I₀
→ I(z_i) = I₀ · exp(-cumtrapz(α, z))

数值实现：用 cumtrapz (cumulative trapezoidal integration)
  α_array = 4π κ(z_array) / (n λ)
  optical_depth = cumtrapz(α_array, z_array, initial=0)
  I_array = I₀ · exp(-optical_depth)

热源：
  q(z_i) = α(z_i) · I(z_i)
```

### 3.3 观测采样

稀疏观测设计：
```
5 个 z 位置 × 4 个 t 时刻 = 20 个观测点
  z_obs = [1, 5, 10, 20, 30] μm  (含缺陷层中心 5μm)
  t_obs = [0.01, 0.05, 0.1, 0.5, 1.0] × 1e-3 s（5 个时刻，实际是 5×4=20）

噪声：1% 乘性高斯噪声
```

观测点选择理由：
- z=1μm：表面，绝热区主导
- z=5μm：缺陷层中心，强吸收（关键观测点）
- z=10μm：缺陷层外，过渡区
- z=20μm：纯基底，验证外推
- z=30μm：远场，验证 BC2

## 4. PINN 架构

### 4.1 双网络设计

```
网络 1: T_θ(z, t) — 温度场
  输入: (z, t), shape [N, 2]
  架构: 3 层 × 64 神经元, tanh 激活
  输出: T(z, t), shape [N, 1]
  参数量: ~8.6k

网络 2: κ_φ(z) — 消光系数场
  输入: z, shape [N, 1]
  架构: 3 层 × 32 神经元, tanh 激活
  输出: κ(z), shape [N, 1], softplus 正值约束
  参数量: ~2.3k
```

### 4.2 损失函数

```
L = λ_data · L_data + λ_phys · L_phys + λ_ic · L_ic + λ_bc · L_bc + λ_prior · L_prior

L_data = MSE(T_θ(z_obs, t_obs), T_obs)              # 20 个观测点
L_phys = MSE(r_phys, 0)                              # 256 个配点
  r_phys = ∂T_θ/∂t - D · ∂²T_θ/∂z² - q(z;κ_φ)/(ρCp)
L_ic = MSE(T_θ(z, 0), 0)                             # 50 个 IC 点
L_bc = MSE(∂T_θ/∂z|_{z=0}, 0)                       # 50 个 BC 点
L_prior = MSE(κ_φ(z), κ₀)                            # 弱先验，防发散
```

权重（初始值，可手动调度）：
```
λ_data = 1.0
λ_phys = 1.0
λ_ic = 1.0
λ_bc = 1.0
λ_prior = 0.01   # 弱，仅防发散
```

### 4.3 自动微分链

```
z, t → T_θ(z, t)
  → autograd.grad(T, t, create_graph=True) = ∂T/∂t
  → autograd.grad(T, z, create_graph=True) = ∂T/∂z
  → autograd.grad(∂T/∂z, z, create_graph=True) = ∂²T/∂z²

z → κ_φ(z)
  → α(z) = 4π κ_φ(z) / (n λ)
  → I(z) = I₀ · exp(-cumtrapz(α, z))   # 数值积分，需要可微实现
  → q(z) = α(z) · I(z)

r_phys = ∂T/∂t - D · ∂²T/∂z² - q(z)/(ρCp)
```

### 4.4 训练

- 优化器：Adam, lr=1e-3
- 步数：5000 步
- 配点：256 个 (z, t) 随机采样（每步 resample）
- 观测点：固定 20 个

## 5. 对比基线

### 5.1 基线 1：解析解 + scipy 最小二乘（均匀 κ 假设）

```python
# 假设 κ 均匀，用 calc_photothermal_timed 做前向
# scipy.optimize.least_squares 反演等效 κ_eff
# 预期：得到一个 κ_eff，无法揭示 κ(z) 分布
# 这是解析解的根本局限
```

### 5.2 基线 2：纯数据驱动 NN（无物理约束）

```python
# 同样 20 个观测点，训练普通 NN T_data(z, t)
# 无 PDE 约束，无 κ 网络
# 预期：观测点附近拟合好，无观测区域完全失败
# 验证：在 (z=15μm, t=0.5ms) 等无观测位置计算误差
```

### 5.3 预期结果表

| 方法 | κ(z) 反演 | T(z,t) 重建（无观测区） | 物理一致性 |
|------|---------|----------------------|:---:|
| scipy 均匀 κ | ❌ 只能给 κ_eff 标量 | ❌ 用错模型 | ❌ |
| 纯数据 NN | ❌ 无法 | ❌ 外推失败 | ❌ |
| **PINN** | ✅ κ(z) 分布 | ✅ 物理外推 | ✅ |

每个组件先写测试，再写实现。测试在 `examples/test_pinn_nonuniform.py`。

### 6.1 测试列表

```
test_kappa_field_design:
  - 真值 κ(z) 在 z=5μm 处达到 4κ₀
  - κ(z) 在 z→∞ 时回到 κ₀

test_finite_difference_solver:
  - 均匀 κ 下 FD 解与 calc_photothermal_timed 误差 < 5%
  - 隐式格式无条件稳定（粗 Δt 不爆）
  - 能量守恒（近似）

  - T_θ(z, t) 输出有限值
  - shape 正确

test_physics_residual:
  - 在解析解已知的情况下（均匀 κ），PINN 物理残差 < 1e-3
  - autograd 高阶梯度计算正确

test_optical_intensity_integration:
  - 均匀 α 下 I(z) = I₀ exp(-αz) 精度 < 1%
  - 非均匀 α 下能量守恒（近似）

test_pinn_training_step:
  - 单步训练 loss 下降
  - 梯度非零

test_baseline_scipy_uniform:
  - scipy 能跑出 κ_eff
  - κ_eff 与真值有显著偏差

test_baseline_pure_nn:
  - 纯 NN 在观测点 loss 低
  - 纯 NN 在无观测点 loss 高

test_run_inversion:
  - 完整 PINN 反演跑通
  - κ(z=5μm) 误差 < 50%（合理预期，PINN 反演函数难度高）
  - T(z,t) 场重建误差 < 10%
```

### 6.2 反演成功判据

由于非均匀 κ(z) 反演是高度病态逆问题，预期精度：
- κ(z) 形状定性正确（5μm 处有峰）
- κ 峰值误差 < 50%（定量）
- T(z,t) 场重建 MSE < 10%
- **关键**：在无观测区域 T(z,t) 误差 < 20%（外推能力）

如果 PINN 反演失败，也要诚实记录，作为案例局限。

## 9. 风险与缓解

| 风险 | 概率 | 缓解 |
|------|:---:|------|
| PINN 训练不收敛 | 中 | 多随机种子 + 学习率调度 + 损失权重调度 |
| κ(z) 反演误差大 | 高 | 接受定性正确，诚实记录；增加观测点 |
| FD 真值生成有 bug | 中 | 先用均匀 κ 与解析解对比验证 FD 实现 |
| autograd 高阶梯度 NaN | 低 | 用 float64 + 梯度裁剪 |
| 训练时间过长 | 中 | 限制 5000 步，CPU 应在 10 分钟内 |
